mysql8.0.11连接驱动

Posted on 2019-02-20 | In 数据库

连接最新版的mysql时报错
异常错误：

Loading class `com.mysql.jdbc.Driver’. This is deprecated. The new driver class is `com.mysql.cj.jdbc.Driver’. The driver is automatically registered via the SPI and manual loading of the driver class is generally unnecessary.

以前版本的jdbc.properties

jdbc.driverClass = com.mysql.jdbc.Driver
jdbc.url = jdbc:mysql://127.0.0.1:3306/db?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
jdbc.username = root
jdbc.password = admin

现在按照最新官方提示支持将 com.mysql.jdbc.Driver 改为 com.mysql.cj.jdbc.Driver

jdbc.driverClass = com.mysql.cj.jdbc.Driver
jdbc.url = jdbc:mysql://localhost:3306/db?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=UTC&rewriteBatchedStatements=true
jdbc.username = root
jdbc.password = admin

mysql8.0.11改时区方法

Posted on 2019-02-20 | In 数据库

第一种

通过修改配置文件my.cnf，进行时区的修改
在文件[mysqld] 之下增加

default-time-zone='+8:00'

没试过，一般不用，公共的mysql一般不重启

第二种

在项目中修改数据库连接地址

jdbc:mysql://XXX:3306/XXX?useUnicode=true&characterEncoding=utf8&autoReconnect=true&failOverReadOnly=false&allowMultiQueries=true&useSSL=true&serverTimezone=Asia/Shanghai

最后的serverTimezone=Asia/Shanghai

hexo博客插入图片方法

Posted on 2019-02-19 | In hexo博客折腾记

  1. 把主页配置文件_config.yml 里的post_asset_folder:这个选项设置为true

  2. 在命令行执行npm install hexo-asset-image --save，安装插件

  3. 插件安装后，执行hexo n "xxxx"生成md博客时，/source/_posts文件夹下会生成一个同名的文件夹

  4. 在md博客中想引入图片时，先把图片复制到同名的文件夹中，然后只需要在博客中中按照markdown的格式引入图片：

(![你想输入的替代文字](xxxx/图片名.jpg))

  1. 最后检查一下，hexo g生成页面后，进入public 对应的html文件中查看相关字段，可以发现，html标签内的语句是

<img src=”****/**/**/xxxx/图片名.jpg”>，
而不是
<img src=”xxxx/图片名.jpg”>。

这很重要，关乎你的网页是否可以真正加载你想插入的图片。

三体I读后感

Posted on 2019-02-18 | In 杂谈

  一直仰慕三体的大名，书在18年618时就已经买了。但是，也许是已经过了看小说的年纪，《三体》一直被我遗忘在角落。直到春节档的《流浪地球》，一下子重燃了我对科幻的热情，也借着这股劲，我翻开了《三体Ⅰ》。
  网上都说《三体Ⅰ》是最难看下去的一本，但是对于我来说，《三体Ⅰ》对我的震撼远大于后两本。这是我第一次接触三体小说，可能巨大的震撼来源于此吧。让我印象最深刻的就是射手假说和农场主假说：

射手假说：有一名神枪手，在一个靶子上，每隔10cm打出一个洞。设想这个靶子上生活着一种二维智能生物，它们中的科学家在对自己的宇宙进行观察后，发现了一个伟大的定律：每隔10cm单位，必然会有一个洞。它们把这个神枪手一时兴起的随意行为，看成自己宇宙中的铁律。

农场主假说：一个农场里有一群火鸡，农场主每天上午11点来给它们喂食。火鸡中的一名科学家观察这个现象，一直观察了近一年都没有出现例外，于是它也发现了自己宇宙中的伟大定律：每天早上11点，就有食物降临。它在感恩节早晨向大家宣布了这个定律，但这天早上11点食物没有降临，农场主进来把他们都捉去杀了。

看到这里，一丝凉意涌上心头。我们世界的法则是真实的吗？还是只是在乱纪元来临之前短暂的恒纪元？亦或者是神的随手一挥，成为了我们坚信的真理，那么下一挥，又会在何时？

  这本书中，我看到了大刘在努力地构建着书中的世界观，如此的真实而又可怕。但是最可怕的不是真理，而是一个个在敌人来临之际愿意毁灭家园的社会精英。对社会的失望，使他们将希望寄托于文明远高于地球的三体。但是，“主”的降临不是希望，而是毁灭，而他们究竟能得到什么？后代的灭亡，与他们无关，他们只需要追求精神的逃避，追求所谓的“理想”。杀人者主也，非我也。错——世界也，非我也。
